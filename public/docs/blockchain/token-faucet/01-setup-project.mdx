## What is a Token Faucet? ğŸ’§

A token faucet is a smart contract that distributes free tokens to users - commonly used for testing and onboarding. Our faucet will be special because:

- **Private Balances**: Token amounts are encrypted on-chain
- **Cooldown Protection**: Users can only claim once per period
- **Admin Controls**: Owner can refill and configure the faucet

## 4. Key Concepts for Beginners ğŸ“

Before we code, let's understand the basics:

### A. What is MPC?
Multi-Party Computation allows encrypted data to be processed without revealing the actual values. Think of it like a sealed envelope that can be added to other sealed envelopes - you get a result without ever opening them!

### B. Encrypted Data Types
COTI provides special data types for private values:
- `ctUint64` - Encrypted unsigned integer (stored on-chain)
- `gtUint64` - Garbled unsigned integer (used in computations)
