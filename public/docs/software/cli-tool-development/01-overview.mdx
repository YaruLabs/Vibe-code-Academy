# CLI Tool Development

You are a **developer tools specialist** with expertise in building developer-friendly CLI applications.

## CLI Tool Requirements

Create a powerful CLI tool with the following features:

### Command Structure

```bash
# Main command with subcommands
mytool <command> [options]

# Examples
mytool init --template react
mytool build --watch
mytool deploy --env production
```

### Architecture

#### Commander.js Example
```javascript
import { Command } from 'commander';

const program = new Command();

program
  .name('mytool')
  .description('A powerful CLI tool')
  .version('1.0.0');

program
  .command('init')
  .description('Initialize a new project')
  .option('-t, --template <type>', 'Project template')
  .action((options) => {
    console.log('Initializing with template:', options.template);
  });
```

## Essential Features

### 1. Interactive Prompts
Use libraries like `inquirer` for user-friendly input:

```javascript
import inquirer from 'inquirer';

const answers = await inquirer.prompt([
  {
    type: 'list',
    name: 'framework',
    message: 'Which framework?',
    choices: ['React', 'Vue', 'Angular']
  },
  {
    type: 'confirm',
    name: 'typescript',
    message: 'Use TypeScript?',
    default: true
  }
]);
```

## Auto-completion

### Bash/Zsh Support
```bash
# .mytool-completion.sh
_mytool_completion() {
  local commands="init build deploy test"
  COMPREPLY=($(compgen -W "$commands" -- "${COMP_WORDS[1]}"))
}

complete -F _mytool_completion mytool
```
